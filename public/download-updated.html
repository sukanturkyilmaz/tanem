Yükleme Adımları:

  2. cPanel File Manager → public_html
  3. ÖNEMLİ: Eski assets dosyalarını silin
  4. index.html dosyasını güncelleyin (üzerine yazın)
  5. Yeni CSS dosyasını assets/ içine yükleyin
  6. Yeni JS dosyasını assets/ içine yükleyin
  7. Eski CSS ve JS dosyalarını silin (isimleri farklı)
