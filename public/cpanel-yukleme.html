⬇️ Paketi İndir

⚠️Yüklemeden ÖNCE Yapılacaklar

  1. Mevcut dosyaları yedekleyin (varsa)
  2. public_html klasörüne gidin
  3. Şu dosyaları silin:
    • index.html
    • assets/ klasörü
    • .htaccess (varsa eski)
    • Diğer tüm eski proje dosyaları

📋Yükleme Adımları

  1. Paketi indirin yukarıdaki butondan
  2. cPanel File Manager'ı açın
  3. public_html klasörüne gidin
  4. Upload butonuna tıklayın
  5. PUBLIC-HTML-READY.tar.gz dosyasını seçip yükleyin
  6. Dosya yüklendikten sonra dosyaya sağ tıklayın
  7. "Extract" seçeneğine tıklayın
  8. Çıkartma tamamlandıktan sonra PUBLIC-HTML-READY.tar.gz dosyasını silebilirsiniz
  9. Tarayıcınızda stnsigorta.com adresini açın ve test edin

🔍Yükleme Sonrası Kontrol

  • public_html içinde index.html olmalı
  • public_html içinde assets/ klasörü olmalı
  • public_html içinde .htaccess olmalı
  • Site açıldığında login ekranı görünmeli

✅Paket İçeriği

Bu paket React uygulamanızın build edilmiş halini içerir. İçinde .htaccess dosyası bulunur (HTTPS yönlendirme ve React Router desteği ile). Hiçbir ekstra konfigürasyon gerekmez.
